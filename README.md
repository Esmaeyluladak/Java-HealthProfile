#  Health Profile & BMI Calculator (Java)

Bu proje, nesne yönelimli programlama (OOP) prensipleri kullanılarak geliştirilmiş bir Java konsol uygulamasıdır. Kullanıcıların boy, kilo ve doğum tarihi bilgilerini alarak detaylı bir sağlık profili oluşturur.

##  Öne Çıkan Özellikler
- **Nesne Yönelimli Yaklaşım:** Veri güvenliği için `private` değişkenler ve `getter/setter` metodları (Encapsulation).
- **Yaş Hesaplama:** `java.time.LocalDate` kullanılarak güncel yıl üzerinden yaş tespiti.
- **BMI Analizi:** Dünya Sağlık Örgütü standartlarına göre kilo durumunun belirlenmesi.

##  Teknik Detaylar
- **Dil:** Java
- **Kullanılan Yapılar:** Classes, Objects, Conditionals, Scanner.

##  Nasıl Çalıştırılır?
1. Bilgisayarınızda Java JDK yüklü olduğundan emin olun.
2. `Solution.java` ve `HealthProfile.java` dosyalarını aynı dizine indirin.
3. Terminalde `javac Solution.java` komutuyla derleyin.
4. `java Solution` komutuyla uygulamayı başlatın.
